## Project Overview

AI-Powered Ticket Triage Agent is an intelligent support ticket management system that automatically analyzes incoming customer tickets using Artificial Intelligence and Machine Learning techniques. The system reduces manual effort by classifying tickets into predefined categories, determining their priority, and generating useful insights for support teams.

The goal of this project is to improve response time, optimize resource allocation, and enhance customer support efficiency through automated ticket analysis.

---

# AI-Powered Explanations

The Ticket Triage Agent uses AI models to understand ticket content and predict appropriate classifications. Natural Language Processing (NLP) techniques are applied to process unstructured text such as customer complaints, technical issues, and service requests.

The AI system performs the following tasks:

* **Ticket Classification:** Identifies the category of the ticket such as technical issue, billing problem, account request, or general inquiry.
* **Priority Prediction:** Determines urgency levels such as High, Medium, or Low.
* **Text Processing:** Cleans and transforms raw text into machine-readable features.
* **Automated Decision Support:** Assists support teams in assigning and handling tickets faster.

---

# Reporting and Dashboard Analysis

The system provides analytical insights into ticket data through reports and dashboards.

Dashboard features include:

* Number of tickets processed.
* Distribution of ticket categories.
* Priority-level analysis.
* Trends in customer issues.
* Model prediction performance.
* Processing success and failure statistics.

Reports help administrators identify common issues, monitor support workload, and improve service quality.

# Setup Instructions

## 1. Clone Repository

```bash
git clone https://github.com/Manasa2530/Ticket-Triage-Agent.git
cd Ticket-Triage-Agent
```

## 2. Create Virtual Environment

```bash
python -m venv venv
```

Activate environment:

Windows:

```bash
venv\Scripts\activate
```

Linux/Mac:

```bash
source venv/bin/activate
```

## 3. Install Dependencies

```bash
pip install -r requirements.txt
```

## 4. Run the Application

```bash
python main.py
```
